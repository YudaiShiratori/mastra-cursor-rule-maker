# Mastra Cursor Rule Maker

このプロジェクトは[Mastraで作るAIエージェント入門](https://zenn.dev/serinuntius/books/4346a0fc6818f3)を参考に開発した、AIエージェント開発フレームワーク「Mastra」を使用したアプリケーションです。

## 概要

Mastraフレームワークを活用して、Cursor IDEのルール作成を支援するAIエージェントを開発するプロジェクトです。

## 技術スタック

- [Mastra](https://mastra.ai/) - AIエージェント開発フレームワーク
- TypeScript
- OpenAI/Google AI SDK

## 始め方

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```
